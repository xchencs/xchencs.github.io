About Me

I received the B.Sc. degree in the School of Management from Xi’an Jiaotong University （西安交通大学）, China, in 2002.

Then I received the M.Sc. and Ph.D. degrees in the Department of Computer Science and Technology from Nanjing University（南京大学）, China, in 2008 and 2011, respectively. My advisor is Professor Daoxu Chen （陈道蓄） and Professor Qing Gu （顾庆）.

Now I am an associate professor in the School of Information Science and Technology at Nantong University.

I am a member of ACM, IEEE and senior member of CCF (China Computer Federation).

Research Interests

My research interests include AI4SE, software analytics, software testing, software maintenance, empirical software engineering, mining software repository:

  • LLMs for Software Engineering (LLM4SE) （Literautre Repository By SMU） （Literautre Repository by NJU）

  • Security Vulnerability Understanding, Detection, Localization, and Repair

  • Code Change Representation and Downstream Tasks (Just-in-time Defect Prediction, Commit Message Generation, Just-in-time Vulnerability Detection)

  • Mining Software Repository (Software Defect Prediction, Stack Overflow Mining, Source Code Summarization, Code /API Search)

  • Software Fault Localization and Automatic Program Repair （Literautre Repository）

  • Regression Testing (Test Case Selection, Test Suite Augmentation, Test Case Prioritization, Test Case Reduction)

  • Combinatorial Testing（Literature Repository）

More research information can be found by visiting our smartSE group （智能软件工程研究组）.
